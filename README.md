# 📊 Projeto de Análise de Vendas com SQL

## 🎯 Objetivo

Este projeto tem como objetivo analisar dados de vendas para identificar padrões de consumo, comportamento de clientes e desempenho de produtos, gerando insights que auxiliem na tomada de decisão.

---

## 🛠️ Ferramentas Utilizadas

* SQL
* (Próximo passo: Power BI)

---

## 🧱 Estrutura do Banco de Dados

O projeto foi estruturado em três tabelas principais:

* **clientes** → informações dos clientes
* **produtos** → dados dos produtos e preços
* **pedidos** → registro das vendas realizadas

O relacionamento entre as tabelas permite analisar o comportamento de compra e calcular métricas de negócio.

---

## 📊 Análises Realizadas

### 💰 Receita Total

Foi calculada a receita total da empresa considerando a quantidade vendida e o preço dos produtos.

**Insight:**
A análise permite entender o faturamento geral e serve como base para acompanhar o desempenho do negócio.

---

### 📦 Produto Mais Vendido

Identificação dos produtos com maior volume de vendas.

**Insight:**
Produtos com maior quantidade vendida indicam maior aceitação no mercado e podem ser priorizados em estoque e campanhas.

---

### 💸 Receita por Produto

Cálculo do faturamento gerado por cada produto.

**Insight:**
Nem sempre o produto mais vendido é o mais lucrativo. Essa análise ajuda a identificar quais produtos realmente geram mais receita.

---

### 👤 Clientes com Mais Pedidos

Análise do volume de pedidos por cliente.

**Insight:**
Clientes recorrentes são responsáveis por uma parcela significativa das vendas, sendo importantes para estratégias de fidelização.

---

### 📅 Receita por Período

Análise da receita ao longo do tempo.

**Insight:**
Permite identificar variações de faturamento e possíveis tendências de crescimento ou queda.

---

## 🧠 Principais Aprendizados

Durante o desenvolvimento deste projeto, foram aplicados conceitos fundamentais de SQL:

* Uso de `JOIN` para relacionamento entre tabelas
* Funções de agregação como `SUM()` e `COUNT()`
* Agrupamento de dados com `GROUP BY`
* Ordenação de resultados com `ORDER BY`
* Criação de métricas de negócio a partir dos dados
